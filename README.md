# Phone Number to Words

A beginner-friendly Python project that converts each digit in a phone number into words.

## How It Works

The user enters a phone number, and the program prints the name of each digit.

For example:

```text
Enter your phone number: 123
One Two Three
```

If the user enters a character that is not a number, the program prints `!` for that character.

## How to Run

Make sure Python is installed on your computer.

Run the program with:

```bash
python main.py
```

On some systems, you may need:

```bash
python3 main.py
```

## Example Output

```text
Enter your phone number: 9870
Nine Eight Seven Zero
```
